Overview

TradeVista is a React + Django based stock portfolio analytics platform designed to help users track investments, analyze market performance, compare assets, and visualize financial trends through interactive dashboards.

The platform combines portfolio management, forecasting, machine learning analytics, Bitcoin prediction, precious metals comparison, and stock comparison into a unified experience.

Key Features

  • Portfolio Tracking Dashboard
  • Stock Analytics & Performance Monitoring
  • Bitcoin Forecasting
  • Gold & Silver Trend Analysis
  • Machine Learning Insights
  • Risk Clustering
  • Portfolio Growth Projection
  • Stock Comparison Engine
